const and scope

A variable (var) name has been created and assigned originally the string "Andrew". However, open DevTools, and you'll see that its value is "Joel "Kraft". This due to the variable being overwritten by another function in the JS script.
To prevent it, just declare it with const!(and also declare the additional name variable inside the function scope as a const variable).

const with arrays and objects

Declaring arrays and objects as const variables does not prevent their contents from being modified with their corresponding methods. It does however prevent you from reassigning it to a new object! See results in Console.
